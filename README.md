# TUTility

豊橋技術科学大学の学生用アプリ(非公認)です。

## 機能

- 時間割取得・表示
- 食堂メニュー表示
- 各種リンク一覧

## 開発

Node.js v22.15.0 で開発しています。

1. 依存関係のインストール

   ```bash
   npm install
   ```

2. 開発サーバーの起動

   ```bash
   npm start
   ```

iOS / Android 端末に「[Expo Go](https://expo.dev/go)」をインストールし、ターミナルに表示されたQRコードを読み取ることで動作確認できます。
